# v0.1.0-experimental.1

Experimental test release for checking Sheepfold installation and the LuCI `Update app` button.

This is not a production-ready release yet.

## Package

Attach this package to the GitHub Release:

`luci-app-sheepfold-family-internet-control_0.1.0-58_all.ipk`

The package uses `Architecture: all` because it contains LuCI assets, shell scripts, UCI config, and rpcd ACL files without native binaries.

## What to test

- Install the `.ipk` manually through LuCI.
- Open Sheepfold in LuCI.
- Go to `Settings` -> `Misc`.
- Press `Update app`.
- Confirm the action.
- Check that Sheepfold can read the latest GitHub Release, download the `.ipk`, and run `opkg install`.

## Notes

- The GitHub Release must be a normal release, not a prerelease, because the updater currently reads GitHub's latest stable release endpoint.
- The release title should clearly say `Experimental test release`.
- The release body should mention that it is intended only for testing update flow.
