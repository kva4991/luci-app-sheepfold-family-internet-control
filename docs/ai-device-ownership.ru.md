# ИИ-анализ принадлежности устройств

> Статус: **исследовательская идея, не реализовано и не готово к прямому кодингу**. Предположение модели о владельце устройства не является фактом и не может автоматически менять личную группу. Перед реализацией свериться с [`ai-assistant-development/`](ai-assistant-development/README.md).

## Цель

Автоматически определить, кому из членов семьи принадлежит каждое устройство в сети, которое ещё не привязано к личной группе. Результат — процентное соотношение вероятности принадлежности по каждому человеку.

## Когда запускается

Функция `sheepfold-ai-ownership` запускается автоматически из `sheepfold-service` если:
- накопилось ≥ 30 событий активности по устройствам **не состоящим** ни в одной личной группе, **и**
- прошло не менее 1 часа с последнего запуска.

Запись последнего запуска: `/tmp/sheepfold/ai-ownership-last-run`.

## UCI: хранение уникального ID устройства и процентов

Каждому устройству присваивается `deviceId` при первом обнаружении — **однажды и навсегда**:

```uci
config device 'device_aabbccddeeff'
    option mac             'AA:BB:CC:DD:EE:FF'
    option deviceId        'dev_7f3a9c'        # sha256(mac)[:6], неизменен
    option personalGroupId ''                  # пусто — не в личной группе
    # проценты принадлежности (обновляет ИИ)
    option ownershipVasya  '72'
    option ownershipMasha  '18'
    option ownershipUnknown '10'
    option ownershipUpdatedAt '1720375200'
```

## Промпт для ИИ (передаётся через CGI → внешний ИИ)

```
SYSTEM:
Ты — помощник по анализу домашней сети. Твоя задача: по журналам активности устройств определить, кому из известных членов семьи принадлежит каждое неизвестное устройство.

Правила:
- Возвращай только JSON. Никаких пояснений вне JSON.
- Для каждого устройства выдавай объект с полем "ownership": { "<имя>": <процент 0-100>, ... }. Сумма процентов = 100.
- Если принадлежность не определяется — используй имя "unknown".
- Устройства из личных групп в анализ НЕ включены — не упоминай их.
- Опирайся на: время активности, тип устройства, паттерны сайтов, hostname, соседство с известными устройствами (одновременная активность).

FORMAT:
{
  "devices": [
    {
      "deviceId": "dev_7f3a9c",
      "mac": "AA:BB:CC:DD:EE:FF",
      "hostname": "android-phone",
      "ownership": { "Вася": 72, "Маша": 18, "unknown": 10 },
      "reasoning": "кратко — почему такое распределение"
    }
  ]
}

USER:
<сюда вставляется JSON с логами и списком членов семьи (имена из личных групп)>
```

## Автоматическое добавление в личную группу

После получения ответа ИИ `sheepfold-ai-ownership` проверяет каждое устройство:
- Если для какого-то человека процент ≥ 85 **и** для этого человека существует личная группа — вызывает `personal-group-add-device <pgId> <mac>`.
- Событие записывается в лог: `"Устройство AA:BB:CC:DD:EE:FF автоматически добавлено в личную группу 'Вася' (уверенность ИИ: 87%)."`
- Уведомление отправляется в мессенджер (Telegram/VK).
