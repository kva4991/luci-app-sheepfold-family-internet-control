# Sheepfold Android Child (приложение ребёнка)

## 1. Назначение

Android-приложение для ребёнка (каталог `android-child`) — лёгкий клиент, который устанавливается на устройство ребёнка и передаёт на роутер информацию о состоянии, может показывать локальные ограничения и уведомления.[cite:80]

## 2. Целевая аудитория

- Дети, использующие устройство под контролем родителей.
- Дизайнеры детского мобильного интерфейса.
- Тестировщики, проверяющие корректность взаимодействия child-приложения с роутером.

## 3. Основные функции

- Передача информации о клиентском статусе на роутер (через `sheepfold-api-client-status`).[cite:60]
- Отображение локальных уведомлений о блокировке/ограничении.
- Возможное отображение таймера/счётчика до конца разрешённого времени.

## 4. Для дизайнеров

### 4.1. Принципы дизайна

- Интерфейс должен быть простым и понятным для ребёнка.
- Минимум управляющих элементов: фокус на отображении статуса и сообщений.
- Сдержанная визуализация ограничений (без излишней тревожности), но с ясностью, что доступ ограничен.

### 4.2. Экраны

- Экран статуса: онлайн/офлайн, доступ разрешён/ограничен.
- Экран сообщения от родителя (если реализовано).
- Экран таймера до конца сессии.

## 5. Для тестировщиков

- Проверка корректности отправки IP/MAC и статуса на роутер.[cite:60]
- Проверка отображения сообщений при разных сценариях (полная блокировка, временный доступ, списки сайтов).
- Проверка работы при отсутствии связи с роутером.

## 6. Связь с другими приложениями

- Child-приложение не управляет правилами, а только отражает их и передаёт статус; управление идёт через роутер и родительское приложение.[cite:80]
