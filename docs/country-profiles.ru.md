# Профили стран

Sheepfold должен использовать профили стран, чтобы адаптировать:

- доступных провайдеров ИИ-помощника;
- стартовые предложения аварийно-полезных сайтов;
- описания доменов;
- предупреждения;
- язык по умолчанию.

## Зачем нужны профили стран

Роутер может быть установлен в разных странах. Полезные сайты, экстренные службы, школьные платформы, банки и доступность ИИ-провайдеров отличаются по стране.

Профили стран не дают зашить российские предположения во весь продукт.

## Правила профиля

- Пользователь выбирает страну нахождения роутера при настройке.
- Пользователь может изменить страну позже.
- Смена страны не должна удалять ручные записи.
- Смена страны может обновить предлагаемые аварийно-полезные сайты, но не должна автоматически включать их без подтверждения родителя.
- Смена страны может изменить список видимых ИИ-провайдеров.
- Юридическую и сетевую доступность провайдеров считать конфигурацией, а не вечной истиной в коде.
- Если статус провайдера неизвестен, показывать его как недоступный, пока он явно не настроен.

## Черновик профиля России

### ИИ-провайдеры

DeepSeek — предпочтительный кандидат по умолчанию там, где он разрешён и доступен.

Профиль должен поддерживать дополнительных провайдеров только если они разрешены и доступны в России.

У записи провайдера должны быть:

```text
id
display_name
api_base_url
default_model
status: allowed | unavailable | prohibited | unknown
notes
```

Стартовый список для России:

| Домен | Название | Описание |
| --- | --- | --- |
| `gosuslugi.ru` | Госуслуги | Портал государственных услуг. |
| `esia.gosuslugi.ru` | Вход через Госуслуги | Единая система авторизации. |
| `mos.ru` | Сервисы Москвы | Городские сервисы для пользователей из Москвы. |
| `school.mos.ru` | Московская школа | Московская электронная школа. |
| `dnevnik.ru` | Электронный дневник | Электронный дневник и школьная коммуникация. |
| `ya.ru` | Простой поиск Яндекса | Минимальная страница поиска Яндекса. |

Не добавлять широкий `yandex.ru` в стартовый список России. См. `AGENTS.md` и `docs/domain-allowlist.ru.md`.

### Транспортные предложения

Профиль страны может показывать транспортные сайты как предложения, а не как автоматически включённый пресет.

Для России можно предлагать к ручному подтверждению:

```text
2gis.ru
rzd.ru
pass.rzd.ru
transport.mos.ru
mosmetro.ru
```

Такси можно показывать только как отдельный ручной транспортный список, не как стартовое предложение:

```text
taximaxim.ru
city-mobil.ru
vezet.ru
taxovichkof.ru
```

Яндекс Go и Яндекс Такси не добавлять в стартовые предложения по умолчанию. Это слишком широкий суперапп-сценарий: такси может тянуть за собой маркетплейс, еду, доставку, каршеринг, самокаты и другие неаварийные функции.

## Будущие профили стран

Каждый профиль страны должен использовать те же критерии, что и Россия:

- государственная идентификация и госуслуги;
- школа и образовательные платформы;
- экстренные и медицинские сервисы;
- карты и транспорт;
- важные банки, если уместно;
- связь только если она не превращает ограниченный режим в обычный интернет;
- по умолчанию не добавлять развлечения, игры, короткие видео, социальные ленты, стриминг, магазины приложений и широкие порталы.

Первые файлы профилей:

```text
country-profiles/ru.json
country-profiles/us.json
country-profiles/de.json
country-profiles/fr.json
country-profiles/es.json
country-profiles/tr.json
country-profiles/ua.json
country-profiles/kz.json
```
