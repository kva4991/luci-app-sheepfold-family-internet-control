# ИИ-профили членов семьи

> Статус: **идея / готово к проектированию**

## Цель

Периодически формировать и обновлять психологический портрет каждого члена семьи: личность, рост, интересы, отношение к другим членам семьи, паттерны поведения онлайн.

Результат хранится в файле на роутере и используется ИИ-помощником при следующем разговоре как контекст.

## Хранение

`/tmp/sheepfold/ai-family-profiles.json` — в RAM, при перезагрузке восстанавливается из резервной копии `/etc/sheepfold/ai-family-profiles.json`.

Структура:

```json
{
  "schemaVersion": 1,
  "updatedAt": 1720375200,
  "members": [
    {
      "personalGroupId": "pg_a1b2c3",
      "displayName": "Вася",
      "role": "child",
      "ageYears": 12,
      "personality": "Любопытный, импульсивный, легко увлекается новым. Хорошо реагирует на логику и примеры из игр.",
      "interests": ["Minecraft", "YouTube", "программирование", "велосипед"],
      "growthNotes": "За последние 2 месяца вырос интерес к программированию. Стал спокойнее реагировать на ограничения.",
      "approachHints": "Объяснять правила через аналогии из игр. Не давить авторитетом — спорит. Хвалить за самоконтроль.",
      "familyRelations": {
        "Маша": "соперничает, но защищает",
```

## Когда обновляется

- Автоматически, не чаще **одного раза в 7 дней** на члена семьи.
- Принудительно — по команде `sheepfold-ai-profiles update <pgId>`.
- При обновлении ИИ получает **предыдущий профиль** и данные за последние 7 дней — он не пересоздаёт с нуля, а дополняет/корректирует.

## Промпт для ИИ

```
SYSTEM:
Ты — семейный психолог-наблюдатель. Твоя задача: обновить психологический профиль члена семьи по данным о его сетевой активности за последние 7 дней.

Правила:
- Не ставь диагнозов. Не делай тревожных выводов из малого числа событий.
- Пиши тепло, без осуждения.
- Поле "approachHints" — конкретные советы родителям: как говорить с этим человеком, на что опираться.
- Поле "familyRelations" обновляй только если есть новые данные (совместная активность устройств, паттерны общего времени онлайн).
- Возвращай только JSON-объект члена семьи (структуру см. в CURRENT_PROFILE).
- Не меняй personalGroupId и displayName.

CURRENT_PROFILE:
<предыдущий JSON-профиль>

USER:
<JSON с активностью за 7 дней: сайты, время, продолжительность сессий>
```

## Связь с ИИ-помощником

При каждом разговоре с ИИ-помощником в системный промпт добавляется краткая выжимка профилей всех членов семьи (поля `personality`, `approachHints`, `interests`). Это позволяет давать **персонализированные советы** и правильно выбирать тон разговора.
