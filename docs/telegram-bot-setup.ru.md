# Настройка Telegram-бота для Sheepfold

Этот документ описывает ручную настройку Telegram-бота для управления Sheepfold. Бот работает через исходящее подключение роутера к Telegram Bot API, поэтому роутеру не нужен белый IP, публичный домен или входящий webhook.

## Что понадобится

- установленный Telegram у родителя;
- доступ к LuCI Sheepfold на домашнем OpenWRT-роутере;
- доступ роутера в интернет;
- 5-10 минут времени.

## Важное про безопасность

- Создавайте отдельного бота только для Sheepfold.
- Не публикуйте токен бота.
- Не отправляйте токен детям, в семейные чаты, в поддержку или в публичные группы.
- Относитесь к токену как к паролю: кто знает токен, тот может управлять ботом.
- Не добавляйте Sheepfold-бота в публичные группы.

## Шаг 1. Найти BotFather

1. Откройте Telegram.
2. В поиске найдите `@BotFather`.
3. Проверьте, что имя именно `@BotFather`.
4. Откройте чат с BotFather.
5. Нажмите `Start` или отправьте команду `/start`.

Официальная документация Telegram по ботам: https://core.telegram.org/bots/tutorial

## Шаг 2. Создать бота

1. Отправьте BotFather команду `/newbot`.
2. BotFather попросит имя бота. Это обычное видимое имя, например:

```text
Sheepfold Home
```

3. BotFather попросит username бота. Он должен быть уникальным и заканчиваться на `bot`, например:

```text
my_sheepfold_home_bot
```

Если username уже занят, придумайте другой.

## Шаг 3. Скопировать токен

После создания BotFather пришлёт токен. Он выглядит примерно так:

```text
123456789:AAExampleTokenExampleTokenExampleToken
```

Скопируйте этот токен в LuCI:

1. Откройте Sheepfold.
2. Перейдите в `Настройки`.
3. Откройте вкладку `Мессенджер`.
4. В поле `Активный мессенджер` выберите `Telegram`.
5. Вставьте токен в поле `Токен Telegram-бота`.
6. Нажмите `Сохранить`.

## Шаг 4. Получить chat ID родителя

1. Откройте созданного бота в Telegram.
2. Отправьте ему любое сообщение, например:

```text
привет
```

3. Если поле `Chat ID родителя-администратора` в Sheepfold пустое, Sheepfold должен ответить в Telegram вашим chat ID.
4. Скопируйте этот chat ID.
5. Вставьте его в поле `Chat ID родителя-администратора`.
6. Нажмите `Сохранить`.

После этого Sheepfold будет принимать команды только от разрешённого ID пользователя, указанного в настройках роутера.

## Шаг 5. Проверить подключение

В LuCI нажмите кнопку `Отправить тестовое сообщение Telegram`.

Если всё настроено правильно, бот пришлёт сообщение:

```text
Sheepfold Telegram подключен к роутеру ...
```

```text
/start
/help
/status
/devices
/internet_on
/internet_off
/wifi_on
/wifi_off
/support
/grant_time #3 30
/block_device #3
/unblock_device #3
/allowlist_add #3
/blocklist_add #3
/logs
/clear_logs
/update
/reboot
/emergency_sites
/wifi_status

помощь
статус
показать все устройства
включить интернет
отключить интернет
выключить интернет
включить вайфай
выключить вайфай
дать #3 30 минут
заблокировать #3

добавить #3 в белый список

журнал

обновить приложение

```

Команды принимаются только от разрешённого ID пользователя, указанного в настройках Sheepfold. Опасные команды (`очистить журнал`, `обновить приложение`, `перезагрузить роутер`) требуют отдельного подтверждения кодом.

## Если не работает

- Проверьте, что на роутере есть интернет.
- Проверьте, что выбран `Telegram`, а не `Выключено`.
- Проверьте, что токен скопирован полностью.
- Проверьте, что chat ID вставлен без пробелов.
- Отправьте боту новое сообщение после сохранения токена.
- Нажмите `Отправить тестовое сообщение Telegram`.
- Откройте журнал Sheepfold и посмотрите последние ошибки.

## Идея для будущих скриншотов

Позже сюда можно добавить собственные скриншоты:

- поиск `@BotFather` в Telegram;
- команда `/newbot`;
- ввод имени бота;
- ввод username;
- сообщение BotFather с токеном;
- вкладка Sheepfold `Настройки -> Мессенджер`;
- ответ Sheepfold с chat ID;
- успешное тестовое сообщение.

Скриншоты лучше хранить локально в репозитории, например в `docs/assets/telegram-setup/`, а не подтягивать с внешних сайтов.
